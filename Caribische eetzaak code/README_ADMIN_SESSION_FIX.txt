ADMIN LOGIN FIX

Probleem:
Op de homepage stond de link altijd naar admin_login.php.
Daardoor leek het alsof je steeds opnieuw moest inloggen.

Oplossing:
- Als admin is ingelogd, gaat de knop naar admin.php.
- Admin login stuurt je automatisch door naar admin.php als je al bent ingelogd.
- Normaal uitloggen logt alleen de klant uit, niet de admin.
- Admin uitloggen gebeurt apart via admin_logout.php.
